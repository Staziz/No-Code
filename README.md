FocusNova

A modern, minimalist web application to help users maintain focus using the Pomodoro technique.

Requirements

A countdown timer starting at 25 minutes.

Start, Pause, and Reset buttons.

A simple text input to add tasks to a 'To-Do' list.

The app must look highly professional, using a modern color palette and smooth animations.

Play a notification sound (using Web Audio API or a generic beep) when the timer hits zero.

Keep track of how many Pomodoro sessions have been completed today.

SYSTEM ARCHITECTURE WARNING

OUTPUT ONLY A SINGLE HTML FILE.
The application MUST be entirely self-contained within index.html.
All CSS must be in a <style> tag.
All logic must be in a <script> tag.
Do NOT output any markdown, explanations, or text outside of the HTML structure.
